## **Установка и запуск ClearML**

### **1. Установка ClearML**

```bash
pip install clearml
```

### **2. Инициализация ClearML**

```bash
clearml-init
```

Вас попросят ввести:

* **API Server URL**
* **Web Application URL**
* **Files Server URL**
* **Access Key / Secret Key**

Все эти параметры можно найти в своём аккаунте на [ClearML](https://app.clear.ml):
**Settings → Workspace Credentials**
